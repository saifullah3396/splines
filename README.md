# Splines
=======================================================

A c++ based implementation of polynomial and b-splines using eigen library.

## Built With
* [EIGEN](https://eigen.tuxfamily.org) - Eigen documentation

## Build
* Build steps are given below:
```
mkdir build && cd build
cmake ..
make
```
